---

## Features

* Built with **Django REST Framework (DRF)**
* **PostgreSQL** integration for reliable data storage
* **JWT authentication** for secure user and admin access
* **CORS enabled** for frontend communication
* **Role-based permissions** for customers and admins
* Separate **API routes** for public and authenticated users

* Banners
